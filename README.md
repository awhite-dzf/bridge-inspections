# bridge-inspections
This project works with bridge data from the Federal Highway Administration (FHWA). The purpose is to see if bridge quality can be modeled and accurately predicted, so that costly and unnecessary bridge inspections can be reduced. The project also creates a priority score for all U.S. bridges calculating which bridge repairs should be prioritized.

Dataset: https://ngda-transportation-geoplatform.hub.arcgis.com/datasets/usdot::national-bridge-inventory/about
Documentation: https://www.fhwa.dot.gov/bridge/mtguide.pdf
